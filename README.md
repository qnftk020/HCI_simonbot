# HCI_simonbot

A Telegram bot that collects papers from major HCI (Human-Computer Interaction) conferences and journals, then provides AI-powered summaries on demand using Google's Gemini model.

## Overview

HCI_simonbot aggregates **42,396 papers** from **20 venues** spanning **2015–2025**, covering the most influential conferences and journals in the HCI field. Users can explore papers randomly, filter by specific venue or publisher, and receive concise Korean-language summaries generated by Gemini 3.1 Flash Lite Preview.

## Data Pipeline

```
1. DBLP API — Collect metadata from 20 venues x 11 years (2015-2025)
   ├── Uses stream: filter for venues with stable DBLP keys
   ├── Uses venue: filter for venues requiring name-based matching
   └── Retrieves: title, authors, year, URL (no abstracts from DBLP)
                    │
2. Semantic Scholar API — Supplement with additional HCI papers
   ├── Keyword search: "human computer interaction",
   │   "user interface design", "usability study", "interactive systems"
   └── Retrieves: title, authors, year, URL, abstract
                    │
3. Semantic Scholar API — Enrich DBLP papers with abstracts
   ├── For each DBLP paper missing an abstract,
   │   search Semantic Scholar by title to find matching abstract
   └── Rate-limited: 3s delay between requests
                    │
4. SQLite DB — Deduplicated storage (by source_id)
   ├── Tagged with: venue, publisher, paper_type (conf/journal)
   └── Indexed on: title, source, venue, publisher
                    │
5. Gemini API — On-demand summarization (user request only)
   ├── Triggered by: /random, /random_venue, /random_pub
   ├── Generates: 3-4 line Korean summary of the abstract
   └── Cached in DB to avoid redundant API calls
```

## Bot Commands

| Command | Description |
|---------|-------------|
| `/start` | Welcome message and command overview |
| `/random` | Get a random paper with AI summary |
| `/random_venue <venue>` | Random paper from a specific venue (e.g., `/random_venue CHI`) |
| `/random_pub <publisher>` | Random paper by publisher (e.g., `/random_pub ACM`) |
| `/search <keyword>` | Search papers by title or abstract |
| `/venues` | List all venues with paper counts |
| `/stats` | Database statistics (totals, abstracts, summaries, breakdowns) |
| `/clear` | Clear cached summaries |
| `/help` | Show help message |

## Setup

### Prerequisites
- Python 3.9+
- Telegram Bot Token (from [@BotFather](https://t.me/BotFather))
- Google Gemini API Key (from [Google AI Studio](https://aistudio.google.com/))

### Installation

```bash
# Clone the repository
git clone https://github.com/qnftk020/HCI_simonbot.git
cd HCI_simonbot

# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Configure environment variables
cp .env.example .env
# Edit .env with your API keys
```

### Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `TELEGRAM_BOT_TOKEN` | Yes | Telegram bot token from BotFather |
| `GEMINI_API_KEY` | Yes | Google Gemini API key |
| `GDRIVE_CREDENTIALS_PATH` | No | Path to Google Drive OAuth credentials |
| `GDRIVE_FOLDER_ID` | No | Google Drive folder ID for DB backup |

### Running

```bash
# Direct execution
python main.py

# With PM2 (recommended for 24/7 operation)
pm2 start main.py --name hci-paper-chatbot --interpreter venv/bin/python
pm2 save
```
